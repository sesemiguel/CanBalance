# Can Balance

### Given Problem


Given a non-empty array, return true if there is a place to split the array so that the sum of the numbers on one side is equal to the sum of the numbers on the other side.


#### Sample Input 
[1, 1, 1, 2, 1]

[2, 1, 1, 2, 1]

[10, 10]


#### Sample Output 
True

False

True
